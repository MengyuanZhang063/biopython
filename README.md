# 关于我们

BioPython是一个开源的**生物/医学背景Python入门和进阶学习社区**，致力于构建一个纯粹、优质的学习圈子，帮助学习者更好的成长。内容覆盖Python编程基础、可视化、组学数据处理、数学基础、机器学习、深度学习、理论应用等。
